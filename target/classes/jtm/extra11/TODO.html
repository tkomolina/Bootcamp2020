Extra 11 exercise: Extra 11: Java 8

  1. Review presentation

      

  2. Ensure you are using Eclipse 4.x.

  3. Change settings in pom.xml file:

    ...
    <!-- Set explicitly Java version for compiler -->
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-compiler-plugin</artifactId>
    <version>3.5.1</version>
    <configuration>
    <source>1.8</source>
    <target>1.8</target>
    </configuration>
    ...

  4. Recompile project and regenerate settings for eclipse:

    mvn clean compile eclipse:eclipse
  5. Review package jtm.extra11 and class RandomPerson in package jtm.activity03.
  6. Add new methods into PersonMatcher interface:
    1. method Stream<RandomPerson> getPersonStream()

    2. default method Stream<RandomPerson> getMatchedPersonStream() with parameters:

      • Stream<RandomPerson> persons,
      • boolean isFemale,
      • int ageFrom,
      • int ageTo,
      • float weightFrom,
      • float weightTo

      This method should return Stream of passed persons filtered out according to passed gender, range of age and range of weight (including).

    3. static method List<RandomPerson> getPersonList(Stream<RandomPerson> persons)
    4. static method PersonMatcher getPersonManager()
    5. Implement default and static methods in PersonMatcher interface.
  7. Create class PersonMatcherImpl which implements PersonMatcher interface and implement unimplemented methods for it:
    1. public constructor PersonMatcherImpl()
    2. public void addPerson(RandomPerson person)
    3. public List<RandomPerson> getPersonList()
    4. public Stream<RandomPerson> getPersonStream()
  8. Check that unit test is passing for jtm.extra11 package.
  9. Check that unit tests are not failing for other packages (can fail if deprecated API of Java 7 was used in some implementation).
